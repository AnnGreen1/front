FLV.js 简介

FLV.js 是一个由 Bilibili 开源的纯 JavaScript 实现的 FLV 播放器，它的工作原理是将 FLV 文件流转换为 ISO BMFF（Fragmented MP4）片段，然后通过 Media Source Extensions 将 MP4 片段提供给 HTML5 video 元素。

主要特性

  • 纯 JavaScript 实现，无需 Flash 插件
  • 支持 HTTP FLV 直播流播放
  • 低延迟，高性能
  • 支持 H.264 和 AAC 编码
  • 兼容现代浏览器
  • 支持 MSE 扩展

基本使用方法

// 检查浏览器是否支持 FLV.js if (flvjs.isSupported()) { // 获取 video 元素 var videoElement = document.getElementById('videoElement'); // 创建 FLV 播放器 var flvPlayer = flvjs.createPlayer({ type: 'flv', url: 'http://example.com/live.stream.flv' }); // 附加到 video 元素 flvPlayer.attachMediaElement(videoElement); // 加载视频 flvPlayer.load(); // 播放视频 flvPlayer.play(); }

使用场景

  • 网页直播应用
  • 在线教育平台
  • 视频监控系统
  • 任何需要播放 FLV 格式视频的 Web 应用
