FLV.js 是什么？

FLV.js 是一个由 Bilibili 开源的纯 JavaScript 实现的 FLV 播放器。它的工作原理是将 FLV 文件流转换为 ISO BMFF（Fragmented MP4）片段，然后通过 Media Source Extensions 将 MP4 片段提供给 HTML5 video 元素。

主要特性

  • 纯 JavaScript 实现，无需 Flash 插件
  • 支持 HTTP FLV 直播流和点播播放
  • 低延迟，高性能
  • 支持 H.264 视频编码和 AAC/MP3 音频编码
  • 兼容现代浏览器（Chrome, Firefox, Safari, Edge）
  • 支持 MSE (Media Source Extensions) 技术
  • 开源且免费使用

基本使用方法

// 检查浏览器是否支持 FLV.js if (flvjs.isSupported()) { // 获取 video 元素 const videoElement = document.getElementById('videoElement'); // 创建 FLV 播放器 const flvPlayer = flvjs.createPlayer({ type: 'flv', url: 'http://example.com/live.stream.flv', isLive: true // 如果是直播流 }); // 附加到 video 元素 flvPlayer.attachMediaElement(videoElement); // 加载视频 flvPlayer.load(); // 播放视频 flvPlayer.play(); }

技术细节

FLV.js 通过 JavaScript 解析 FLV 格式，提取其中的音视频数据，然后通过 Media Source Extensions API 将数据传递给 HTML5 video 元素进行播放。这种方式使得在不支持 Flash 的现代浏览器中播放 FLV 内容成为可能。

FLV.js 特别适合低延迟直播场景，因为 HTTP-FLV 协议相比于 HLS 具有更低的延迟特性。
