起步代码

只需几行就能初始化一个地图

app.js 
// 1. 创建地图，设置视图中心和缩放级别
const map = L.map('map').setView([35.86, 104.19], 5);

// 2. 加载瓦片图层（OpenStreetMap 免费底图）
L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
  attribution: '© OpenStreetMap contributors',
  maxZoom: 19
}).addTo(map);

// 3. 添加一个标记并绑定弹窗
L.marker([39.9042, 116.4074])
  .addTo(map)
  .bindPopup('<b>北京</b><br>中国的首都')
  .openPopup();

// 4. 监听地图点击事件
map.on('click', (e) => {
  L.marker(e.latlng).addTo(map)
    .bindPopup(`点击位置: ${e.latlng.lat.toFixed(4)}, ${e.latlng.lng.toFixed(4)}`);
});

核心 API 速查

掌握这些就覆盖了 90% 的日常使用场景

方法 说明
L.map('id').setView([lat, lng], zoom) 初始化地图，指定容器 ID、中心坐标和缩放级别
L.tileLayer(url, options).addTo(map) 加载瓦片图层，常用 OpenStreetMap、高德、Mapbox 底图
L.marker([lat, lng]).addTo(map) 在指定坐标添加标记点，支持拖拽（draggable: true）
layer.bindPopup(html) 绑定弹窗内容，点击标记时弹出 HTML 信息
L.circle([lat, lng], { radius }) 绘制圆形区域，可设置半径（米）、颜色、透明度
L.polyline(latlngs, options) 绘制折线路径，适合路线、边界等
L.polygon(latlngs, options) 绘制多边形区域，支持自动闭合
map.flyTo([lat, lng], zoom) 带动画飞行到目标位置，比 setView 更流畅
map.on('event', callback) 监听事件：click、zoomend、moveend 等
map.removeLayer(layer) 移除已添加的图层 / 覆盖物
L.layerGroup([layers]) 将多个图层编组，便于统一管理和批量操作
